* Early init (loaded before init.el)
Disable package.el at startup so straight.el is the only package manager; avoids the "straight.el was loaded when package.el was already loaded" warning.
#+begin_src emacs-lisp :tangle .emacs.d/early-init.el :mkdirp yes
(setq package-enable-at-startup nil)
#+end_src

* Bootstrap
** Straight.el
Bootstrap straight.el first — it is the sole package manager. Package.el is disabled in =early-init.el=.
#+begin_src emacs-lisp
  (defvar bootstrap-version)
  (let ((bootstrap-file
	 (expand-file-name
	  "straight/repos/straight.el/bootstrap.el"
	  (or (bound-and-true-p straight-base-dir)
	      user-emacs-directory)))
	(bootstrap-version 7))
    (unless (file-exists-p bootstrap-file)
      (with-current-buffer
	  (url-retrieve-synchronously
	   "https://raw.githubusercontent.com/radian-software/straight.el/develop/install.el"
	   'silent 'inhibit-cookies)
	(goto-char (point-max))
	(eval-print-last-sexp)))
    (load bootstrap-file nil 'nomessage))
#+end_src

** Use-package
Install use-package via straight.el and make straight the default backend for all =use-package= declarations.
#+begin_src emacs-lisp
  (straight-use-package 'use-package)
  (setq straight-use-package-by-default t)
  (setq use-package-always-defer t)
  (setq use-package-verbose t)
#+end_src

* Load modules
Add the lisp directory to the load path and load each module.
#+begin_src emacs-lisp
  (add-to-list 'load-path (expand-file-name "lisp" user-emacs-directory))

  (load "core")
  (load "programming")
  (load "org-config")
  (load "files")
  (load "macos")
  (load "extras")
#+end_src

* Auto-tangle on save

Automatically tangle Org config files when saved. Uses =org-auto-tangle= — add =#+auto_tangle: t= to any Org file that should auto-tangle.
#+begin_src emacs-lisp
  (use-package org-auto-tangle
    :defer t
    :hook (org-mode . org-auto-tangle-mode))
#+end_src

* Cleanup
Dial down GC thershold so it runs frequently but in less time.
#+begin_src emacs-lisp
  ;; Make gc pauses faster by decreasing the threshold.
  (setq gc-cons-threshold (* 5 1024 1024))
  (setq read-process-output-max (* 1024 1024)) ;; 1mb
#+end_src
